# Samburu TTS Project

This repository contains all resources, notebooks, and scripts for improving the Samburu text-to-speech model.

## Structure
- `notebooks/` - Colab-ready templates
- `src/` - Helper Python scripts
- `data/` - Placeholder for datasets

## Quickstart
Upload this repo to GitHub and open notebooks directly in Google Colab.

**Base Model:** Facebook MMS-TTS (Samburu)
